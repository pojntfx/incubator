## Usage

```bash
# Install dependencies #1
npm install
# Install dependencies #2
npm run bootstrap
# Start inidividual package
npm run dev -- --scope @pojntfx-incubator/coffeescript-react
# Start all packages
npm run dev
```

## Deployment

```bash
# Publish packages to npm
npm run publish
```
